Q1: Why might we consider using Git?

Answer: [Git lets you track every change you make to your code, so you can always go back to earlier versions if you mess up. Multiple people can 
work on the same thing, and it is stored and backed up safely.]


Q2: What field could data science be applied to? Give an Example.

Answer: [Data science can be applied in many fields, for example it can be used in healthcare. We can use it to analyze
patient records in order to predict diseases earlier. An example can be Building a machine learning model 
that analyzes X-rays to detect pneumonia faster and more accurately]
